五、推荐安装步骤

1. conda create -n clased-ov python=3.10 -y
2. conda activate clased-ov
3. pip install torch==2.4.0+cu121 torchvision==0.19.0+cu121 --index-url https://download.pytorch.org/whl/cu121
4. pip install -U openmim
5. mim install mmengine==0.10.7
6. mim install mmcv==2.2.0
7. pip install mmsegmentation==1.2.2
8. pip install -r requirements.txt

说明：mmcv 带有 CUDA/C++ 扩展，若直接 pip 安装失败，优先使用 OpenMMLab MIM 安装匹配版本。

六、可选依赖说明

当前 CLaSeD-OV 的评估入口主要使用 eval.py、clased_ov_segmentor.py、custom_datasets.py、custom_mass_transforms.py 和 sam3/model 相关推理代码。

保留的 sam3/agent、sam3/train、sam3/eval 子目录中包含上游 SAM 3 的可选代理、训练和视频评测代码，这些路径可能需要 openai、submitit、torchmetrics、hydra-core、omegaconf 等额外依赖。它们不是当前 CLaSeD-OV 评估/推理流程的必要依赖，因此未写入主 requirements.txt。
